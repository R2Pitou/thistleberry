How this is supposed to work

You are on the published website. This page is not just the output. It is also the visual editing surface for site/index.html.

  1. Click Edit this page.
  2. Change the text directly on the page.
  3. Paste a GitHub token with write access to the repository.
  4. Save. The page commits the updated site/index.html back to the repo.

What you need once

Create a GitHub fine-grained token with Contents: Read and write access to this repository, or use a classic token with repo scope. The token is used from your browser to call the GitHub API directly.

What Thistleberry is now

Thistleberry is a thin visual layer on top of a flat HTML page. GitHub remains the source of truth. The browser overlay just helps you edit the page visually and write the updated HTML back to the repository.
