Start here

You created a new site from the template. The homepage source lives in .gitscribe/pages/index.json and the public output lives in site/index.html.

Thistleberry is the translation layer between visual editing and final HTML. Edit the structured source through the CMS app or change the JSON directly if you prefer.

Repository setup

  1. Open this repository on GitHub.
  2. Go to Settings then Pages.
  3. Under Build and deployment, set Source to GitHub Actions.
  4. Push to main and wait for the Deploy Pages Site workflow to finish.
